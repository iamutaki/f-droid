Personal F-Droid repository for experimental and production Android apps.

## Add to F-Droid

1. Open **F-Droid** → Settings → Repositories
2. Scan QR or tap **+** and paste the URL:

   ```text
   https://iamutaki.github.io/f-droid/repo?fingerprint=8DB384FF37887528A99440A47A5AFFC0315CBBEABBD9990509F5F2CB2307601A
   ```

3. Save - F-Droid will fetch the index
